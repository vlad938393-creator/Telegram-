Telegram Bot «Разговор с писателем»
Интеллектуальный Telegram-бот, имитирующий живой диалог с Александром Сергеевичем Пушкиным. Пользователь может читать биографию, знакомиться с произведениями, а также задавать вопросы и получать ответы в стиле писателя.

Возможности
💬 Общение с писателем в режиме диалога
📖 Просмотр биографии и произведений
📝 Чтение стихов и отрывков
✍️ Генерация ответов в стиле автора с помощью модели GPT
Быстрый старт
Склонируйте репозиторий и установите зависимости:
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
Создайте файл .env на основе env.example и заполните:
TELEGRAM_BOT_TOKEN — токен бота из BotFather
OPENAI_API_KEY — ключ доступа к API
OPENAI_MODEL — имя модели, например gpt-5.0-nano
DATABASE_PATH — путь до базы (по умолчанию data/literature.db)
Инициализируйте базу данных (таблицы и данные создаются автоматически при первом запуске).
Запустите бота:
python -m src.main
Структура проекта
src/ — исходный код бота
data/ — база данных и материалы
env.example — пример конфигурации окружения
requirements.txt — зависимости проекта
Замечания
Все секретные данные храните в .env, не коммитьте его в репозиторий.
Чтобы добавить новых авторов, расширьте сиды в модуле базы данных.
Для вопросов к писателю используется комбинация заранее подготовленных ответов и генерации через LLM.
